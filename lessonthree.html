    Lesson 3: Adding CSS Style to HTML Structure

    Avoiding Repetition

    Avoiding repetion helps programmers save time, avoid errors, and make adjustments quickly.

    Avoiding repition helps programmers save time by not having to write every style detail for each occurrence. Instead it allows programmers to write code for one group of elements that will have the same format throughout the web page.

    Avoiding repition helps programmers avoid errors by allowing them to put less code than they would if they had to write every style attribute for each element. The more often programmers have to write minor details, the more chances there are for errors and the harder it is to find and correct those errors.

    Avoiding repetiton also allows programmers to make adjustment to their code easily by not having to go through every line to make changes.

    CSS

    CSS stands for Cascading Style Sheets. The cascading part is the most important to remember because it's what allows programmers to avoid repetition. Essentially, CSS gives programmers a way to apply style to groups of elements of HTML documents by grouping similar elements into classes that are then given the same styles. This grouping method is a huge time saver and simplifier because it helps programmers avoid repetition.
